# PREZ

> Petit prompt, grande présentation.

Génère des présentations HTML pédagogiques avec Claude.

## Démarrage

```bash
npm install
npm run dev
```

## Utilisation

1. Se connecter (email ESIG)
2. Coller sa clé API Claude
3. Décrire la présentation
4. Télécharger le HTML

## Raccourcis (présentation)

| Touche | Action |
|--------|--------|
| ↓ → Espace | Suivante |
| ↑ ← | Précédente |
| C | Contraste élevé |

## Configuration

```env
ALLOWED_EMAILS=steve@esig.ch
```
